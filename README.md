## Voyage Atelier — Luxury Travel Content Architect

Voyage Atelier spins elevated social media narratives for luxury travel brands in seconds. Select a destination, choose the travel signature, dial the tone, and receive a ready-to-publish campaign kit—signature caption, platform adaptations, story arc, visual shot list, and hashtag suite.

### Features
- Destination intelligence spanning Amalfi, Kyoto, St. Barts, Marrakech, Patagonia, and Seychelles
- Travel style matrix across yacht residencies, palace takeovers, wellness sanctuaries, adventure expeditions, and chef-led circuits
- Brand voice controls for minimalist, concierge, insider, and editorial personalities
- Platform-specific playbooks for Instagram, TikTok, LinkedIn, and Pinterest
- One-click copying for individual modules or the entire campaign canvas

### Local Usage
This project is a static HTML/CSS/JS experience—no build step required.

```bash
# optional: preview locally
npx serve .
```

Then visit the printed URL (defaults to `http://localhost:3000`).

### Deployment
Optimized for Vercel static hosting via `vercel.json`. Deploy with:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-fe3522e4
```

### Stack
- Vanilla HTML, CSS, and JavaScript
- Google Fonts (Inter & Playfair Display)
- Zero runtime dependencies for lean deployments
